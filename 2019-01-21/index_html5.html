HTML5의 semantic tag(의미있는 태그)
HTML5 이전에는 구역을 나누어 사용할 경우 DIV tag를 이용해서 설정을 하고, id나 class를 부여해서 모양을 꾸미기 했다.
HTML5부터는 tag를 의미있는 단어를 사용해서 설정을 하고 id나 class를 부여해서 모양 꾸미기를 한다.
  • header : 본문의 제목을 표시하는 영역
  • section : 본문의 큰 영역
  • article : 본문의 문단 영역
  • footer : 본문의 꼬리말 영역
